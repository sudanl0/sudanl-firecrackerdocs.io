Verify that the given CPU template is applied as intended.

This function is an arch-agnostic part of CPU template verification. As template formats differ between x86_64 and aarch64, the arch-specific part converts the structure to an arch-agnostic HashMap implementing ModifierMapKey before calling this arch-agnostic function.
