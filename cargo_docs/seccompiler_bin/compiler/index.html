Module defining the logic for compiling the deserialized filter objects into the IR. Used by seccompiler-bin.

Via the Compiler::compile_blob() method, it also drives the entire JSON -> BLOB transformation process.

It also defines some of the objects that a JSON seccomp filter is deserialized into: Filter, SyscallRule. The rest of objects are deserialized directly into the IR (intermediate representation): SeccompCondition, SeccompAction, SeccompCmpOp, SeccompCmpArgLen.
