pub struct Tap {
    tap_file: File,
    pub(crate) if_name: [u8; 16],
}
Expand description

Handle for a network tap interface.

For now, this simply wraps the file descriptor for the tap device so methods can run ioctls on the interface. The tap interface fd will be closed when Tap goes out of scope, and the kernel will clean up the interface automatically.
