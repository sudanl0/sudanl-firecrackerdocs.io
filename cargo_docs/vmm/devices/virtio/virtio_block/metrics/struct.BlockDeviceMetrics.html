pub fn aggregate(&mut self, other: &Self)

block metrics are SharedIncMetric where the diff of current vs old is serialized i.e. serialize_u64(current-old). So to have the aggregate serialized in same way we need to fetch the diff of current vs old metrics and add it to the aggregate.
