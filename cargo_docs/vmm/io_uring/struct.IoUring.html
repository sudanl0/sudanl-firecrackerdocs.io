Create a new instance.

Arguments
  • num_entries - Requested number of entries in the ring. Will be rounded up to the nearest power of two.
  • files - Files to be registered for IO.
  • restrictions - Vector of Restrictions
  • eventfd - Optional eventfd for receiving completion notifications.

pub unsafe fn push<T: Debug>( &mut self, op: Operation<T> ) -> Result<(), (IoUringError, T)>

Push an Operation onto the submission queue.

Safety

Unsafe because we pass a raw user_data pointer to the kernel. It’s up to the caller to make sure that this value is ever freed (not leaked).

source

pub unsafe fn pop<T: Debug>(&mut self) -> Result<Option<Cqe<T>>, IoUringError>

Pop a completed entry off the completion queue. Returns Ok(None) if there are no entries. The type T must be the same as the user_data type used for push-ing the operation.

Safety

Unsafe because we reconstruct the user_data from a raw pointer passed by the kernel. It’s up to the caller to make sure that T is the correct type of the user_data, that the raw pointer is valid and that we have full ownership of that address.
