fn format_imds(json: &Value) -> Result<String, Error>

Returns the serde::Value in IMDS format plaintext. Currently, only JSON objects and strings can be IMDS formatted.

See the docs for detailed description of the IMDS format: https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/ec2-instance-metadata.html

Examples
{
    "key1" : {
        "key11": "value11"
        "key12": "value12"
    }
    "key2" : "value3"
    "key3" : "value3"
}

IMDS formatted JSON object:

key1/
key2
key3

JSON string:

"value"

IMDS formatted string:

value

If the serde_json::Value is not supported, an UnsupportedValueType error is returned.
